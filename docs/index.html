Correctify

A simple, cross-platform grammar correction app powered by OpenAI. Bring your own API key, pay only for what you use, and keep your data private.

Download Now View on GitHub

What It Does

Correctify corrects grammar, spelling, and punctuation while preserving your writing style and formatting. It's a minimal, distraction-free menubar app that works on macOS, Windows, and Linux.

Features

Menubar App

Lives in your system tray/menubar for instant access. Single click to show/hide. No dock icon on macOS. Launch at startup option available.

Writing Style Selector

Choose from 5 different writing styles: Grammar Only, Formal, Informal, Collaborative, or Concise. Your preference persists across sessions.

Privacy First

Your API key stays local and is never logged. No text storage, no database, no analytics or tracking. Your data remains yours.

Global Shortcut

Correct text from any app using Cmd+Shift+. (macOS) or Ctrl+Shift+. (Windows/Linux). Fully customizable to your preference.

Powered by OpenAI

High-quality corrections using GPT-4o Mini, GPT-5 Mini, or GPT-5. Choose the model that fits your needs. Pay-per-use pricing with no subscriptions.

Sound Notifications

Optional audio feedback for all notifications with distinct sounds for different notification types. Enable/disable in Settings.

Getting Started

1

Get an OpenAI API Key

Visit the API keys page on OpenAI, sign in or create an account, and generate a new secret key.

2

Configure Correctify

Open the app from your system tray/menubar, click Settings, and paste your API key. Optionally enable launch at startup and customize your keyboard shortcut.

3

Start Correcting

Type or paste text in the app, or use the global shortcut (Cmd/Ctrl+Shift+.) from any application to correct text instantly.

Requirements

OpenAI API Key

Get one at platform.openai.com/api-keys

Internet Connection

Required to reach OpenAI's API for text corrections
